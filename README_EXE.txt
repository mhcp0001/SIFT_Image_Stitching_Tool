================================================================================
  SIFT Image Stitching Tool - EXE版
================================================================================

PyInstallerでビルドされた単一実行ファイル版です。

================================================================================
【特徴】
================================================================================

✅ Pythonインストール不要
✅ 単一の.exeファイルで動作
✅ ダブルクリックで即起動
✅ ブラウザが自動的に開く
✅ すべての依存関係が含まれる

================================================================================
【システム要件】
================================================================================

- OS: Windows 10/11 (64bit)
- メモリ: 4GB以上推奨
- ディスク容量: 200MB以上の空き容量

================================================================================
【使い方】
================================================================================

1. SIFT_Stitcher.exe をダブルクリック

2. コンソールウィンドウが開き、サーバーが起動します

3. ブラウザが自動的に開きます
   （開かない場合は手動で http://127.0.0.1:5000 にアクセス）

4. Webインターフェースで画像をアップロードして処理

5. 終了するにはコンソールウィンドウを閉じるか、Ctrl+C を押す

================================================================================
【ファイル構成】
================================================================================

起動すると、exeファイルと同じフォルダに以下が作成されます：

SIFT_Stitcher.exe のあるフォルダ/
├── uploads/          アップロードされた画像の一時保存先
└── results/          合成結果の保存先

================================================================================
【ファイルサイズ】
================================================================================

SIFT_Stitcher.exe: 約92MB

大きなサイズの理由:
- OpenCV (画像処理ライブラリ)
- NumPy/SciPy (数値計算ライブラリ)
- Flask (Webサーバー)
- scikit-image (画像処理)

これらのライブラリがすべて1つのexeに含まれています。

================================================================================
【注意事項】
================================================================================

⚠️ 初回起動時の注意:
   - 初回起動時は展開処理のため、起動に10-20秒かかります
   - 2回目以降は通常の速度で起動します
   - ウイルス対策ソフトが警告する場合がありますが、安全なファイルです

⚠️ アンチウイルスソフトについて:
   - PyInstallerで作成されたexeは、一部のアンチウイルスソフトが誤検出することがあります
   - Windows Defenderの除外リストに追加することを推奨します

⚠️ ポート使用について:
   - このアプリケーションはポート5000を使用します
   - 他のアプリケーションがポート5000を使用している場合、起動に失敗します

================================================================================
【ビルド方法（開発者向け）】
================================================================================

このexeファイルは以下のコマンドでビルドされています：

  build_exe.bat

または

  python -m PyInstaller SIFT_Stitcher.spec

必要な環境:
- Python 3.11以上
- PyInstaller 6.17以上
- すべての依存パッケージ（requirements.txt参照）

================================================================================
【トラブルシューティング】
================================================================================

Q: exeをダブルクリックしても起動しない
A: 以下を確認してください：
   - ウイルス対策ソフトがブロックしていないか
   - ファイルパスに日本語や特殊文字が含まれていないか
   - 管理者権限で実行してみる

Q: ブラウザが自動的に開かない
A: 手動で http://127.0.0.1:5000 を開いてください

Q: 「ポート5000は既に使用されています」エラー
A: 他のアプリケーションがポート5000を使用しています
   - そのアプリケーションを終了するか
   - タスクマネージャーでポート5000を使用しているプロセスを終了

Q: 処理が遅い
A: 以下を確認してください：
   - 画像サイズが大きすぎないか（1枚あたり10MB以下推奨）
   - メモリが不足していないか
   - パラメータ設定が適切か
