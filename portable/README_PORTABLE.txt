================================================================================
  SIFT Image Stitching Tool - ポータブル版
================================================================================

このフォルダには、Pythonのインストール不要で動作するポータブル版が含まれています。

================================================================================
【初回セットアップ】
================================================================================

1. setup_portable.bat をダブルクリックして実行してください

   このスクリプトは以下の処理を自動的に行います：
   - Python 3.11.9 埋め込み版のダウンロードと展開
   - pipのインストール
   - 必要なパッケージのインストール（OpenCV, Flask, NumPy等）

   ⚠️ 注意事項：
   - インターネット接続が必要です
   - 初回セットアップには5-10分程度かかります
   - ウイルス対策ソフトが警告する場合がありますが、安全なスクリプトです

2. セットアップが完了すると「セットアップが完了しました！」と表示されます


================================================================================
【アプリケーションの起動】
================================================================================

1. start_portable.bat をダブルクリックして実行してください

2. ブラウザで以下のURLを開いてください：

   http://127.0.0.1:5000

3. Webインターフェースが表示されます


================================================================================
【使い方】
================================================================================

1. 広角画像（Overview）を1枚選択
   - ドラッグ&ドロップまたはクリックで選択

2. クローズアップ画像を複数選択
   - 複数の高解像度画像を選択可能

3. パラメータを調整（オプション）
   - Canvas Scale: 出力倍率（1-5倍）
   - Blend Strength: ブレンド強度（11-71）
   - SIFT Min Matches: 最小マッチング数（8-30）

4. 「合成を開始」ボタンをクリック

5. 処理が完了したら「結果をダウンロード」ボタンで保存


================================================================================
【フォルダ構成】
================================================================================

portable/
├── README_PORTABLE.txt        このファイル
├── setup_portable.bat         初回セットアップスクリプト
├── start_portable.bat         アプリケーション起動スクリプト
├── python_embed/              Python埋め込み版（初回セットアップ時に作成）

├── uploads/                   アップロードされた画像の一時保存先

================================================================================
【システム要件】
================================================================================

- OS: Windows 10/11 (64bit)
- メモリ: 4GB以上推奨（画像サイズによる）
- ディスク容量: 500MB以上の空き容量
- インターネット接続（初回セットアップ時のみ）


================================================================================
【トラブルシューティング】
================================================================================

Q: setup_portable.bat を実行してもエラーが出る
A: 以下を確認してください：
   - インターネット接続が有効か
   - ウイルス対策ソフトがブロックしていないか
   - 管理者権限で実行してみる

Q: start_portable.bat でサーバーが起動しない
A: setup_portable.bat を先に実行してください

Q: ブラウザでページが表示されない
A: 以下を確認してください：
   - サーバーが起動しているか（コマンドプロンプトが開いているか）
   - URLが http://127.0.0.1:5000 で正しいか
   - ファイアウォールがブロックしていないか

Q: 画像の合成に失敗する
A: 以下を確認してください：
   - 画像フォーマットがJPEG/PNGか
   - 画像が破損していないか
   - SIFT Min Matchesを下げてみる（例: 8）
